# operator-thesis.md

## Core belief

Google Ads is not primarily a bidding problem.
It is an **intent interpretation problem**.

If unlike intents are mixed together, every downstream decision gets worse:
- bids optimize against noise
- RSAs become generic
- landing pages blur the offer
- negatives get missed
- good queries subsidize bad ones

## The operator stance
1. find signal before polishing dashboards
2. cut waste before chasing tiny optimizations
3. separate unlike intent into separate decision buckets
4. use buyer language from actual queries
5. preserve what the account has learned

## The Query Interview
Every serious Google Ads review should ask:
- What is the user actually trying to do with this search?
- Is this search likely to buy, compare, learn, or bounce?
- Should this query live in the same optimization bucket as the others?
- What language is repeated by converters?
- What language is repeated by wasted spend?
- What structural change does this pattern imply?
