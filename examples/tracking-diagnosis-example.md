# Example: Tracking Diagnosis

> Before you optimize anything, the agent answers the question nobody asks: *is the data trustworthy?*

## Can we trust the account?
Mostly.

## Why
- Primary conversion is a real business outcome: booked demo.
- But GA4 import and native Google Ads tag both appear to count the same demo completion.
- Value logic is too flat to support strong value-based bidding decisions.

## Risks
- Duplicate counting may be flattering CPA.
- Value signals are too weak for confident scale decisions.
- Some recent performance gains may be measurement artifacts.

## What decisions should wait
- Aggressive scaling
- Strong conclusions about target CPA efficiency
- Any value-based bidding upgrade

## What can still be learned safely
- Search term quality
- Query clusters
- Brand vs non-brand contamination
- Comparison intent behavior
